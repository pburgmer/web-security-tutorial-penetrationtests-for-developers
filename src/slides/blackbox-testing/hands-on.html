Hands-on: Setup
  1. Kali Linux starten (root:toor) (Host-Only-Network)
  2. ZAP starten
  3. ZAP Port auf 9090 ändern (Tools / Options / Local Proxy)
  4. Iceweasel (Firefox unter Debian) für ZAP konfigurieren
  5. OWASP WebGoat Standalone runterladen und starten
  6. ISO Images für 'Web for Pentesters' und 'Web for Pentesters II' runterladen
  7. Leere Virtuelle Maschine anlagen (Host-Only-Network, 1GB RAM)
  8. Live OS von ISO Image in VM starten
  9. IP Adressen der Virtuellen Maschinen in Host-Only-Network ermitteln (ifconfig)
